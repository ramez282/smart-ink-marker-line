Applied Mechatronics Project

Smart Ink Marker Manufacturing Production Line Simulator

A full-stack industrial automation simulator for assembling ink markers with a Python FastAPI backend, React HMI dashboard, InfluxDB metrics, Grafana visualization, and Docker Compose deployment.

View Project Evidence GitHub Repository

Core Capabilities

Features

Industrial HMI

Start, Stop, and Reset controls with live machine state, production stage, alarms, and metrics.

Quality Control

Rule-based quality score with PASS/FAIL status and clear defect explanations.

SCADA-Style Events

Simple event log for machine actions, quality failures, warnings, and maintenance signals.

Grafana Metrics

InfluxDB stores production metrics that can be visualized in Grafana dashboards.

Manufacturing Flow

Production Line Stages

01

Tip Feeding

Feeds and places the marker tip.

02

Ink Filling

Fills the marker with a target volume of approximately 5.0 ml.

03

Body Assembly

Checks alignment of the marker body components.

04

Cap Fitting + QC

Checks cap force and makes the final PASS/FAIL decision.
